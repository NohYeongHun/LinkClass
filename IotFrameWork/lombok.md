# lombok
1. @Data
- getter(), setter(), toString()등 편리한 기능 사용 가능.

2. @AllArgsConstructor
- 생성자 만들어줌. => 개수가 있는 생성자
- ex) abc (int a, string b, float c)

3. @NoArgsConstructor
- 기본 생성자 만들어줌.

## 설치순서
- lombok.jar 파일을 설치받는다.
- 해당 폴더(다운로드 받은 폴더)에서 cmd(터미널)를 실행시킨다.
- 'java -jar lombok.jar' 실행
- 이후 사용하는 이클립스에 install
