2. 계수만을 이용한 풀이
- 연립일차방정식을 첨가행렬로 나타내어 풀이
```
X + 2Y = 3
-X -3Y = -1
A = [1 2 3; -1 -3 -1];
rref(A)
ans =
1 0 7
0 1 -2
X = 7 / Y = -2
영상 예제
syms x y
S1 = solve('4*x+y=6','2*x+y=4')
S1 = [S1.x, S1.y]

수정된 코드
syms x y
S1 = solve(4*x+y==6, 2*x+y==4)
S1 = [S1.x, S1.y]
```
